# 🏥 Smart Hospital Resource Management System

A relational database mini-project designed to efficiently manage hospital operations, including patient records, staff management, appointments, billing, and resource allocation.

---

## 📌 Project Overview

This project implements a **Smart Hospital Database System** using SQL and a Flask-based web interface.
It centralizes hospital data and ensures efficient handling of resources, reducing manual effort and improving data consistency.

---

## 🚨 Problem Statement

Hospitals often face:

* Inefficient resource management
* Scattered patient data
* Lack of real-time tracking
* Poor coordination between departments

---

## ✅ Solution

Our system provides:

* Centralized database for all hospital data
* Real-time management of patients, staff, and resources
* Integrated modules for appointments, billing, and prescriptions
* Structured and normalized schema for better performance

---

## 🧩 Features

* Patient Management
* Doctor & Staff Management
* Appointment Scheduling
* Billing System
* Prescription & Medicine Tracking
* Lab Screening Records
* Room Allocation
* Insurance Management

---

## 🗂️ Database Design

* 12+ tables (Patient, Doctor, Nurse, Appointment, Bill, etc.)
* Proper use of **Primary Keys** and **Foreign Keys**
* Fully normalized (up to 3NF/BCNF)
* Ensures data integrity and reduces redundancy

---

## ⚙️ Technologies Used

* **Backend:** Python (Flask)
* **Database:** PostgreSQL
* **Frontend:** HTML, CSS, JavaScript

---

## 📊 Dataset Summary

* 12+ tables
* 250+ records
* Covers patients, appointments, billing, and hospital resources

---

## 🔍 Key Concepts Used

* Joins (Inner, Left Join)
* Nested Queries
* Constraints (PK, FK, NOT NULL, UNIQUE, CHECK)
* Normalization (1NF → BCNF)

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/hospital-management.git
```

2. Navigate to project folder:

```bash
cd hospital-management
```

3. Install dependencies:

```bash
pip install flask psycopg2
```

4. Setup PostgreSQL database:

* Create database `hospital_db`
* Import SQL schema

5. Run the application:

```bash
python app.py
```

6. Open in browser:

```
http://localhost:5000
```
